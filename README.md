# Canadian-Health-Care-Analysis

### Project Overview
Analyzed the distribution of healthcare facilities across Canadian provinces and cities using Power BI, developing DAX measures to track facility counts, provider distribution, and facility types, and building interactive dashboards to uncover regional disparities, concentration patterns, and infrastructure insights for decision-making.

### Tools Used
- Power BI (DAX, Power Query,Data modelling)
